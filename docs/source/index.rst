********************************
Netshow Cumulus Linux Provider
********************************

The Netshow Cumulus Linux Provider is a plugin for the `Netshow, a linux troubleshooting
toolkit <http://netshow.readthedocs.org>`_

This site contains the API for this provider. For more details on how Netshow
works, see the `Netshow App Intro <http://netshow.readthedocs.org/en/latest/index-netshow.html>`_ and `Netshow Library Intro <http://netshow.readthedocs.org/en/latest/index-netshow-lib.html>`_
